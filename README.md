# Temperatura
Sensor de temperatura y humedad por rs485
Material necesario:
 - 2 Arduino micro
 - 2 módulo convertidor MAX485 RS-485 TTL 5V
 - 2 módulos dht11
 - 2 led de alta luminosidad
 - 2 resistencias 390 ohm
 - 1 buzzer
 - 1 lcd 2x16
 - 1 interfaz I2C/SPI
 - 2 cajas de montaje.
 - Cables y herramientas varias.
